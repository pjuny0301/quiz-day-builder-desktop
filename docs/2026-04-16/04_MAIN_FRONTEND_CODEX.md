# Main Frontend Codex

상태: 전용 문서
기준일: 2026-04-16

## 역할
- 프론트엔드 화면 구조와 UI 통합을 직접 담당한다.
- 최종 사용자 흐름, 라우팅, 레이아웃, 컴포넌트 연결을 책임진다.
- 다른 Codex 결과를 최종 통합한다.
- 자기 작업에 대해 별도 커밋 허가/거절 리뷰를 남긴다.

## 기본 입력 문서
- `docs/2026-04-15/01_MANDATORY_EXECUTION_ORDER.md`
- 이 문서
- 현재 작업 날짜 폴더의 작업 문서

## 주 책임
- 화면 경계 분리
- UI 결합도 감소
- 라우트/공개 API 경계 유지
- 실제 화면 기준 스모크 테스트와 캡처 검증
- 퀴즈 앱 화면 기준 스크린샷 디자인 일관성 개선
- builder 변경은 quiz flow를 방해하지 않는 범위에서만 처리
- 빌더보다 학습 런처, 학습 진행, 완료 화면을 먼저 맞춘다
- 다른 Codex 결과 승인/거절 판단

## 하지 말 것
- 상태/퀴즈로직 내부 설계를 불필요하게 대신 떠안지 않는다.
- 다른 Codex 전용 문서를 기본으로 읽지 않는다.
