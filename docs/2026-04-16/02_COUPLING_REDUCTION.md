# Coupling Reduction

상태: 시행 중
기준일: 2026-04-16

## 목표
- 코드 A와 코드 B를 동시에 수정하지 않으면 깨지는 결합을 줄인다.
- 파일명이나 상대경로 변경 때문에 연쇄 수정이 생기는 문제를 줄인다.

## 이번 적용 방법
- `alias import`를 도입해 긴 상대경로 의존을 줄인다.
- `public API index.ts`를 두어 바깥 파일이 내부 구현 파일명을 직접 알지 않게 한다.
- 화면은 기능 공개 API만 바라보고, 기능 내부 파일 재배치는 feature 내부에서 흡수한다.

## 이번 패스의 기준
- 모든 파일을 한 번에 바꾸지 않는다.
- 진입점, 라우트, 학습 세션처럼 결합이 큰 축부터 우선 적용한다.
