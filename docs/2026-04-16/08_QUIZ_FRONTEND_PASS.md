# Quiz Frontend Pass

상태: 보류
기준일: 2026-04-16

## 목적
- 퀴즈 화면 전면 리디자인 시도를 기록으로 남긴다.
- 현재 기준에서는 학습 화면이 이미 정리된 것으로 보고, 이 문서를 즉시 구현 기준으로 사용하지 않는다.

## 보류 사유
- 이 문서는 과거 비교 문서를 바탕으로 작성됐다.
- 현재 앱 상태와 사용자의 최신 판단은 `퀴즈 화면 대수술보다 주변 구조 정리 우선`에 가깝다.
- 따라서 학습 런처, 학습 진행, 완료 화면을 다시 크게 뜯는 작업은 일단 보류한다.

## 현재 판단
- 퀴즈 화면은 유지한다.
- 남은 프론트엔드 작업은 `Deck/Day/카드 편집`, `앱 분리`, `전반적 일관성`, `세부 polish` 중심으로 진행한다.
