# Read First

이 폴더의 문서는 선택 사항이 아니다.

강제 규칙:
- 새 작업을 시작하기 전에 이 날짜 폴더 문서를 먼저 읽는다.
- 이후 작업은 `01_MANDATORY_EXECUTION_ORDER.md`의 순서를 어기지 않는다.
- 문서 폴더 명명 규칙은 `YYYY-MM-DD`로 고정한다.
- 새 문서는 가능하면 해당 날짜 폴더 안에 둔다.
- 퀴즈관리앱보다 안드로이드용 퀴즈앱을 우선한다.
- 메인 에이전트는 프론트엔드를 맡는다.
- 구현 담당 하위 Codex는 상태관리/퀴즈로직을 맡는다.
- 심사는 메인 에이전트 또는 별도 리뷰 단계가 맡는다.
- 구현 담당 하위 Codex와 심사 단계를 혼동하지 않는다.
- 하위 Codex 응답은 즉시 소모하지 않고, 커밋 단위 심사 시점에 읽어 토큰 사용을 아낀다.
- 커밋 전 최소 스모크 테스트를 반드시 한다.
