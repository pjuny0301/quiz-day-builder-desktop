# Agent Mandatory Read

이 폴더는 이 프로젝트의 **에이전트 작업 기준 문서 폴더**다.
코드 수정, 리팩터링, 디자인 변경, 버그 수정, 빌드, 검증 모두 이 폴더 문서를 먼저 읽은 뒤 진행한다.

## Read Order

1. `01_IMPLEMENTATION_BASELINE.md`
2. `02_PROJECT_MANAGEMENT_PLAN.md`

## Non-Negotiable Rules

- 구현 전에 위 문서를 먼저 읽고 현재 사용자 방향을 다시 확인한다.
- 함수마다 목적을 설명하는 짧은 한 줄 주석을 바로 위에 단다.
- 급한 땜질보다 확장 가능한 구조 정리를 우선한다.
- 한 화면에는 핵심 역할 하나를 강하게 둔다.
- 값 설정은 가능한 전용 화면에서 처리한다.
- UI 변경은 코드와 빌드만 보지 말고 **스크린샷으로 검증**한다.
- 새 요구가 생기면 문서부터 갱신하고, 그 문서를 기준으로 구현한다.
- 카드 편집은 단일 카드 편집 화면을 유지하고, 대량 입력은 별도 화면에서 처리한다.
- 문서에 `시행`, `미시행` 상태를 남겨 실제 반영 여부를 추적한다.
- 현재 앱은 Windows 데스크톱 고정 크기 기준으로 설계하고, 그 안에서 화면이 안정적으로 보여야 한다.

## Scope

- 이 폴더 문서가 현재 작업의 권위 기준이다.
- 기존 `docs/*.md` 문서는 참고 이력으로 남길 수 있지만, 구현 기준은 이 폴더 문서가 우선한다.
