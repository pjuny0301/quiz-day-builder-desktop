단일 메인 창 안에서 화면 전환으로 동작하는 Deck Manager
1. 덱은 보기 좋은 사각형 카드 그리드로 보임.
2. Deck 카드를 일반 클릭하면 Deck 상세화면으로 넘어감.
3. Deck 카드 오른쪽 위 `...` 메뉴에서 `덱 편집하기`, `덱 삭제하기`를 선택할 수 있음.

`덱 편집하기`를 누르면 넘어가는 수정화면
수정화면에서는 renaming할 수 있고, 지연시간을 설정할 수 있음.
또한 카드를 Split할 수 있고 대표 이미지를 넣을 수 있음.

4. Deck 카드를 눌러 넘어간 Deck 상세화면
넘어간 Deck 상세화면에서는 Day의 목록이 있음.
각 Day는 얇은 리스트가 아님.
시각적으로 굉장히 보기 좋게 되어있음.
각 Day를 표시하는 텍스트는 굵고 큼
Day 에는 퀴즈를 한 번 이상 했을 시 최근 몇 개 중에 몇개 맞았는지 표시되어있음.
이 화면에서는 전체 학습을 시작할 수 있음.
Day를 클릭하면 퀴즈 -> 정답매칭 상세화면으로 넘어감

5. Day를 눌러 넘어간 Day 상세화면
각 퀴즈->정답 매칭은 얇은 리스트가 아님.
시각적으로 굉장히 보기 좋게 되어있음.
각 퀴즈->정답 매칭를 표시하는 텍스트는 굵고 큼
이 화면에서도 4. 와 마찬가지로 play할 수 있음.


6. Study Session은 다시 데스크톱 버전으로 만든다.

7. 화면이 늘어나거나 줄어들거나 상관없게 디자인 구성
